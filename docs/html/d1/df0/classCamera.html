◆ sphericalCoordinateUpdate()

void Camera::sphericalCoordinateUpdate ( )
protected

Calculate spherical coordinates from current camera coordinates.
\( sphericalCameraPosition.x = radius = r = \sqrt{x^{2}+y^{2}+z^{2}} \)
\( sphericalCameraPosition.y = inclination = \theta = \arccos{\frac{y}{r}} \)
\( sphericalCameraPosition.z = azimuth = \varphi = \arctan{\frac{z}{x}} \)
.

Attention
The camera z axis show upward!
In this point we use the origo shifted camera coordinates \( origoCameraPosition = cameraPosition - lookedPoint \)
Built in zero division prevention.
